# Bending Moment and Shear Force Diagrams for a Beam Under Uniformly Distributed Load (UDL)

This project provides a Python implementation for computing and visualizing the **Shear Force Diagram (SFD)** and **Bending Moment Diagram (BMD)** for a **simply supported beam** subjected to a **uniformly distributed load (UDL)** along its entire span.

The script uses **NumPy** for numerical computation and **Matplotlib** for plotting, making it a simple and effective tool for structural engineering students and engineers who want to visualize internal force diagrams.
